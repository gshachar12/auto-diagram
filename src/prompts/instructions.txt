2. Absolute Relative Paths & Icon Syntax (STRICT)
- You MUST use: "../assets/".
- ICON RULE: 'icon' is a TOP-LEVEL property. 
- MANDATORY QUOTES: Every file path MUST be wrapped in double quotes. 
- EXAMPLE: 
  Attacker: "Attacker Client" {
    icon: "../assets/attacker.png";
  }

---
3. Participant-Only Columns (STRICT)
- Standard columns represent network participants ONLY.
- STRIKE RULE: Do not use 'shape: callout' or 'near' keys.
- Use tooltips
---

4. Syntax Bulletproofing (THE "NO-FILL" RULE)
- MANDATORY PREFIX: You MUST use 'style.fill' for all background colors.
- NO EMPTY COLONS: Every colon MUST be followed by a quoted value.
- BLOCK START RULE: The '{' must be on the same line as the definition, preceded by a space.
- HEX CODE RULE: All hex codes (e.g., "#e53935") MUST be enclosed in double quote, like this: style.stroke: "#fb8c00"; 
- Comments are made using #
- Use number emojis like STEP 3️⃣ STEP 4️⃣ to number steps
- "stroke-dash" is a number between 0 and 10
- "stroke-dash" must be style.stroke-dash
Correct Edge Syntax:
Victim_Resolver -> Victim_Resolver: "STEP 4️⃣ **Internal Lookup**" {
    style.stroke: "#fb8c00"; 
    style.fill: "#fff3e0";
    tooltip: "Processing logic here.";
}

5.
LOOP & PHASE NESTING RULE (STRICT)

- Phase blocks group RELATED EDGES ONLY:
  Phase_I: { ... }

- All edges that belong to a phase (including loop/restart steps) MUST be written INSIDE that Phase block.

- NEVER reference phases using prefixes:
  INVALID: Phase_X.A -> Phase_X.B
  VALID:   A -> B   (inside Phase_X block)

- NEVER place edges (especially loop/restart edges) outside the phase they belong to.

- Participants MUST remain GLOBAL (top level).
  Do NOT nest participants inside phases.

- A Phase block may contain ONLY:
  - edges
  - edge-attached style blocks
  - tooltips
  - label and style.fill

- REQUIRED SYNTAX:

  Phase_I: {
    label: "STEP 1"
    style.fill: "#hex"

    A -> B: "STEP 2" {
      style.stroke: "#hex"
      tooltip: "..."
    }
  }

- Use '\n' every 30–35 characters in labels to keep layout narrow.
